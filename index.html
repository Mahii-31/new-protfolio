Tower Game Using Unity

This project involved developing a tower defense game where the player defends their base from waves of enemies. The game uses C# programming and 3D assets from the Unity Asset Store. A key feature of the game is its use of the Depth-First Search (DFS) algorithm to calculate the optimum path for the enemies, adding a strategic challenge for the player to place crossbows along the enemy's path. The game’s design emphasizes engaging gameplay, requiring players to think strategically and adapt to increasingly difficult levels.

IIFC(SOCIAL MEDIA APP)

This web application combines image forgery detection with face recognition to ensure the cyber safety of individuals' credentials. The project focuses on detecting whether an individual's credentials have been forged or morphed. The application provides tools to verify the authenticity of images, making it possible to remove or confirm the originality of the content. The technology behind this project uses Convolutional Neural Networks (CNNs) to perform both forgery detection and face recognition, offering a robust solution to digital image security.
